# Virtual Painter

A Computer Vision project that allows users to draw on a virtual canvas using hand gestures. Built with Python, OpenCV, and MediaPipe, the application tracks hand movements in real time and converts them into digital drawings.

## ✨ Features

* 🖐️ Real-time hand tracking
* 🎨 Multiple brush colors
* 📏 Adjustable brush size
* 🧽 Eraser tool
* ↩️ Undo functionality
* ↪️ Redo functionality
* 💾 Save drawings as images
* 📷 Webcam-based interaction
* ⚡ Smooth and responsive drawing experience

## 🛠️ Tech Stack

* Python
* OpenCV
* MediaPipe
* NumPy


## 🎮 Controls

| Action            | Description                   |
| ----------------- | ----------------------------- |
| Draw              | Move index finger             |
| Selection Mode    | Raise index and middle finger |
| Change Color      | Select from toolbar           |
| Adjust Brush Size | Use brush size controls       |
| Erase             | Select eraser tool           |
| Undo              | Undo last action              |
| Redo              | Restore undone action         |
| Save              | Save drawing as image         |
